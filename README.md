# Numerical Solution for the Present Model

This repository provides the Wolfram Mathematica notebook used to reproduce the finite-element numerical simulations reported in the manuscript.

## File

- `Numerical Solution for the Present Model.nb`: Mathematica notebook for the numerical solution of the present model.

## Software

The simulations were implemented in Wolfram Mathematica using `NDSolve` with finite-element spatial discretization.

## Reproducibility Notes

The notebook contains the Mathematica implementation used for the numerical results discussed in the manuscript. The finite-element discretization is specified in the Mathematica code, including the mesh-related options used in the numerical implementation.

## How to Run

1. Download `Numerical_Solution_for_the_Present_Model.nb`.
2. Open the notebook in Wolfram Mathematica.
3. Evaluate the notebook cells in order to reproduce the numerical solution.
